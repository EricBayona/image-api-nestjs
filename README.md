# Image API - NestJS

API RESTful para autenticación y gestión de imágenes.

## Tecnologías

- NestJS
- Prisma
- PostgreSQL
- JWT
- Multer
- React (frontend separado)

## Features

- Registro y login
- Autenticación con JWT
- Rutas protegidas
- Subida de imágenes
- Almacenamiento local
- Persistencia en PostgreSQL
- Relación usuario / imágenes

## Instalación

```bash
npm install
npm run start:dev
```
